# 帕沫的FF14工具包
一个简单的FF14工具包，用于nonebot，支持真寻bot，实际上是只是分别调用了肥肥咖啡和universalis的接口。查看指令帮助的指令是：/ff help

目前还没有自定义选项，如果想自定义的话可以自行修改源码中的回复词

因为不太会python所以写的有些简陋

食用方法：直接clone然后丢到nonebot或真寻bot对应的的plugins文件夹然后重启bot即可
